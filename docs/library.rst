*************************************
Aqua: 量子アルゴリズム ライブラリー
*************************************

Aquaは、ハイレベルの古典的な計算ソフトウェアの問題を量子コンピューター（シミュレーターまたは本物の量子デバイス）
にマッピングすることを可能にする、エンドツーエンドのクロスドメイン量子ソフトウェアスタックです。
